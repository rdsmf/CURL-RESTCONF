OTHER INTERESING NETWORK PROGRAMABILITY TOOLS
=============================================

1.	ACI SDK - COBRA - (Python SDK that streamlines the process of making API calls to the ACI fabric)
2.	Managed Object Browser - Visore - (GUI Browser of Managed Object hierachy)
3.	API Inspector - (ACI GUI API Inspector)
4.	APIC REST to pYthon Adapter (known as ARYA) - (Python libraries to convert XML / JSON into python code)
5.	ACI ToolKit - (python libraries that allow basic configuration of the Cisco APIC)
6.	API Optimizer - (ACI Simulator using configuration templates)
7.	ACI Power Tool - (manage, operate, and automate Cisco ACI environments)
